# Plasticsearch

## 의미
- plastic: 온순한, 모양을 이루는, 성형의
- search: 검색
- 사실은 [elasticsearch](https://www.elastic.co/elasticsearch/) 모조품

## 정의
- [Apache Lucene](https://lucene.apache.org/) 기반의 **매우 간단**하게 구현된 검색 API 서버
